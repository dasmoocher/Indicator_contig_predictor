Metagenomic indicator contig predictor

Using short & long reads to identify representative contigs as metagenome classifiers

View on GitHub

Abstract

Next Generation Sequencing has redefined the practice of genomic science. In this project, we combine the strength of both “long-read” and “short-read” technologies to be able to discriminate between different sets of microbiomes. For a test case we use male vs female infants, as well as for vaginal versus caesarean births. It is possible that our tool might be able to distinguish between healthy and diseased microbiomes in the future.
